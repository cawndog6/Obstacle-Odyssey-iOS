The Canvas Prefab allows for a simple main menu system to be ported!!
1.Has Simple Start/Exit Button
2. Allows for Exiting the application if Exit is selected, and 
changing the scene to a desired scene on Play being Clicked

Setting Up the Prefab....
* To change the scene that is selected on clicking "Start" 
1. Go to MainMenuControl.cs and navigate to void Play_Clicked
2. Change the Name "Lobby" to the name of the scene you wish to 
	change to.
3. Save the file and test to make sure it changes!
